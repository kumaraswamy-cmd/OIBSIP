# AeroGlass Pro — Tomorrow, in sight.

A premium, Apple-inspired landing page for **AeroGlass Pro**, a fictional next-generation augmented reality (AR) smart glasses product. Built as part of a Web Development and Designing internship project.

---

## 🌟 Key Features

* **Sleek Apple Aesthetics**: Immersive dark mode styled with high-contrast premium typography, subtle gradients, and glassmorphic panels.
* **Responsive Bento Grid**: Modern multi-dimensional CSS Grid highlighting specifications (Displays, processing chipsets, materials, and controls) that reflows gracefully on mobile viewports.
* **Custom Inline SVGs**: Scalable and lightweight graphics including a detailed headset mockup wireframe and vector design components, avoiding heavy external assets.
* **3D Spatial Desktop Simulation**: Fully interactive floating glass dashboard mockups created with pure CSS 3D transforms (`perspective`, `rotateY`, `translateZ`) representing the spatial workspace of **AeroOS**.
* **Micro-Animations**: Keyframe floating animations, pulsing sensor nodes, audio sound wave bars, and glowing chip highlights.
* **Zero Element Overlaps**: Strict viewport controls, percentage pads, and layout-stacking media queries preventing component collisions on mobile, tablet, and desktop screens.

---

## 🛠️ Built With

* **HTML5**: Structured with semantic tags (`<header>`, `<main>`, `<section>`, `<footer>`) and optimized for accessibility.
* **CSS3**: Custom design tokens, CSS Grid, Flexbox, keyframe animations, and 3D transforms.
* **SVG**: Scalable vector graphics.

## 🚀 How to View Locally

This project is a standalone static site. You do not need any compilation or build setups to run it!

### Option 1: Direct Run (Easiest)
1. Clone or download this repository.
2. Locate the `index.html` file in your explorer.
3. Double-click **`index.html`** to open it immediately in any web browser.

### Option 2: Live Server (If Node.js is installed)
We have included a `package.json` setup with Vite for local hot-reloading:
```bash
# 1. Install dependencies
npm install

# 2. Run local dev server
npm run dev
```
